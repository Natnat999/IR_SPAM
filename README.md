# 📡 IR SPAM - Universal Infrared Attacker

![Material 3](https://img.shields.io/badge/Design-Material_3-blue?style=for-the-badge)
![Android](https://img.shields.io/badge/Android-14-green?style=for-the-badge)
![GitHub Actions](https://img.shields.io/badge/Build-Automated-orange?style=for-the-badge)

**IR SPAM** est une application Android moderne (Material You) qui transforme votre téléphone en une télécommande universelle capable d'envoyer massivement des ondes infrarouges pour contrôler (ou éteindre) n'importe quel appareil à proximité.

## 🚀 Fonctionnalités

- **🔥 Mode SPAM** : Envoie en boucle les codes de mise hors tension pour tous les appareils de la catégorie sélectionnée (TV, Clim, Projecteurs, etc.).
- **🔍 Recherche Ultra-Rapide** : Trouvez instantanément votre marque parmi des centaines d'appareils chargés.
- **📱 Material 3 Design** : Support complet des couleurs dynamiques (Material You) et d'un mode sombre élégant.
- **📂 Immense Base de Données** : Samsung, LG, Sony, Daikin, Bose, Apple TV, Roku, et bien d'autres !
- **🛠️ Support Pronto Hex** : Intégration facile de nouveaux codes personnalisés.

## 📦 Télécharger l'APK

Vous pouvez télécharger la dernière version compilée directement dans la section [Releases](https://github.com/Natnat999/IR_SPAM/releases) de ce dépôt.

## 🛠️ Installation & Compilation

Si vous voulez modifier le code ou compiler vous-même :

1. Clonez le dépôt : `git clone https://github.com/Natnat999/IR_SPAM.git`
2. Ouvrez le projet dans **Android Studio**.
3. Compilez sur votre appareil (un émetteur Infrarouge physique est requis sur le téléphone).

## 🦾 GitHub Actions

Chaque commit sur la branche `master` ou chaque création de `tag` déclenche automatiquement :
- La compilation de l'APK de Debug.
- La mise en ligne de l'APK en tant qu'artefact ou Release.
